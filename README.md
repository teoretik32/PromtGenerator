# Prompt Constructor (HTML + CSS + JavaScript)

Веб-конструктор промптов для генерации изображений в GPT / Grok / Nano Banana Pro.

## Что улучшено

- Разбивка монолитного `index.html` на отдельные файлы:
  - `index.html` — структура страницы
  - `styles.css` — стили
  - `prompt-data.js` — база тегов и пресетов
  - `app.js` — логика конструктора
- Расширены группы тегов по ТЗ:
  - национальность, возраст, фигура
  - внешность, эмоция, поза
  - одежда, белье/купальник
  - стиль рендера (фото / аниме / 3D / пиксель)
  - камера/свет/эффекты
  - локация и время суток
  - атмосфера
  - параметры генерации
- Добавлены:
  - свободный ввод
  - быстрые пресеты
  - случайная сборка
  - копирование текстового промпта
  - JSON-вывод + копирование JSON
  - поиск по тегам
  - авто-оценка адекватности промпта
  - кнопка «Сбалансировать» для снижения конфликтов стиля

## Запуск

Откройте `index.html` в браузере.

Если нужен локальный сервер:

```bash
python3 -m http.server 8000
```

Затем перейти на `http://localhost:8000`.

## Идея структуры промпта

Сборка идет блоками:

1. Identity (типаж, возраст, фигура)
2. Face/Look (внешность и эмоция)
3. Clothing (одежда/белье)
4. Scene (поза, стиль, камера, место, атмосфера)
5. Custom text
6. Params (`--ar`, `--v`, `--style raw` и т.д.)

Это позволяет быстро собирать вариации описаний и удобно масштабировать набор тегов.
